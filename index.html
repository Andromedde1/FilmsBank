    Нужна настройка Firebase

    Для общего банка на GitHub Pages подключите Firebase Firestore:

    1. Создайте проект на console.firebase.google.com
    2. Включите Firestore Database
    3. Добавьте Web-приложение и скопируйте конфиг в js/firebase-config.js
    4. Опубликуйте правила из файла firestore.rules
    5. Запушьте изменения в GitHub — сайт обновится автоматически
